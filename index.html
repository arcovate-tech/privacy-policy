1. Introduction

MeetSnap ("we", "our", or "us") is a Chrome extension that helps generate meeting summaries, action items, and follow-up emails from Google Meet captions. We are committed to transparency, minimal data usage, and protecting user privacy.

By using MeetSnap, you agree to this Privacy Policy.

2. Data Collection and Usage

2.1 Meeting Transcripts

MeetSnap only processes data when you explicitly click "Start Capturing".

  • Captions are read directly from the Google Meet page
  • Stored temporarily using Chrome local storage
  • Never transmitted to our servers
  • Sent to Groq API only when you click "Generate Follow-Up"
  • Automatically deleted after processing

2.2 Local Usage Data

  • Number of summaries generated
  • Subscription status (free/paid)
  • User consent confirmation

This data remains entirely on your device.

2.3 Payment Processing

Payments are handled securely by Gumroad. We do not collect or store payment details. See Gumroad Privacy Policy.

2.4 Data We Do NOT Collect

  • No browsing history
  • No audio or video recording
  • No background data collection
  • No data from websites other than meet.google.com

Email address: When you use the Pro verification feature, you enter your purchase email in the side panel to unlock unlimited summaries. This email is sent to our verification system (Supabase) solely to confirm your Pro license. It is not used for marketing, tracking, or any other purpose, and is not stored beyond the verification check.

3. Permissions Justification

MeetSnap requests only the minimum permissions required:

  • scripting and host permissions (meet.google.com): To securely inject the capture script strictly into meet.google.com to read the live captions
  • storage: To temporarily store transcript and usage data locally
  • tabs: To identify the active Google Meet tab, send messages between extension components, and manage tab-specific actions required for capturing and processing meeting captions
  • sidePanel: To display the generated summaries and interface within Chrome's side panel

No permissions are used for tracking or background surveillance.

4. Third-Party Services

Groq API (AI Processing)

  • Used only when user clicks "Generate Follow-Up"
  • Processes transcript to generate summaries
  • Does not retain or store user data
  • Does not use data for AI training

See Groq Privacy Policy.

Gumroad (Payments)

  • Handles all billing securely
  • No payment data is accessible to us

5. User Control and Consent

  • Data processing only starts after user action
  • User must confirm participant consent before capture
  • Users can stop capturing at any time

6. Data Security

  • All communication uses HTTPS encryption
  • No centralized storage of user data
  • Local-only data storage architecture

7. Data Retention

  • Transcript data is deleted immediately after summary generation
  • No long-term storage of any user content

8. User Rights

Since we do not store your data:

  • Uninstalling the extension removes all data
  • You can clear data via Chrome extension settings

9. Children's Privacy

MeetSnap is designed for general professional use and is not specifically targeted at children. We do not knowingly collect personal data from children. If you believe that a minor has used this extension in a way that raises concerns, please contact us and we will take appropriate action.

We may update this policy periodically. Continued use implies acceptance of updates.
