Introduction

PrepSnap ("we", "our", or "us") is a Chrome extension developed by Arcovate Tech that helps users prepare for job interviews by analysing job descriptions and generating AI-powered interview questions, company culture snapshots, and preparation checklists.

We are committed to full transparency, minimal data collection, and protecting your privacy at every step. This policy explains exactly what happens to your data when you use PrepSnap.

Version 1.0 of PrepSnap is completely free. There are no subscriptions, no payments, and no premium tiers in this version. All features are available to every user at no cost.

By installing and using PrepSnap, you agree to this Privacy Policy.

Data Collection and Usage

2.1 Job Description Content

PrepSnap processes job description content only when you explicitly click "Generate Interview Pack." Nothing is collected passively or in the background.

  • Text is read from what you paste, highlight on a page, or link via Google Docs — never collected automatically
  • Sent over HTTPS to our backend (a Supabase Edge Function) only on your explicit action
  • Forwarded to the Groq API solely to generate your interview pack
  • Never stored in any database or log after the response is returned
  • Discarded immediately once the AI response is delivered to your browser

2.2 Google Docs / Drive Links

If you provide a Google Docs or Drive link, our backend fetches the plain text content of that document server-side. This only works with documents set to "Anyone with the link can view." The link and document content are processed entirely in memory and never stored anywhere.

2.3 Local Device Data

The following data is stored only on your device using Chrome's local storage API. It never leaves your device and is never transmitted to any server:

  • Your dark / light theme preference
  • Any text temporarily held between the right-click context menu action and the side panel opening (cleared immediately after use)
No usage counters, no session tracking, no identifiers of any kind are stored locally or remotely.

2.4 Payments — Not Applicable in v1.0

PrepSnap v1.0 is entirely free. There is no payment system, no subscription, and no billing of any kind in this version. We do not collect payment information, email addresses for purchase verification, or any financial data.

If a paid tier is introduced in a future version, this Privacy Policy will be updated accordingly before any payment functionality is enabled, and users will be notified via an updated version number and effective date.

2.5 Data We Do NOT Collect

  • No browsing history or visited URLs
  • No passive or background data collection of any kind
  • No data from websites you visit beyond what you explicitly provide
  • No audio, video, or screen recording
  • No personally identifiable information (name, email, IP address)
  • No advertising IDs, tracking pixels, or behavioural profiling
  • No analytics or crash reporting SDKs

Permissions Justification

PrepSnap requests only the four permissions strictly necessary to function. No permission is used for tracking, surveillance, or any purpose beyond what is described below.

PermissionWhy it is needed
`activeTab` Allows the extension to open the Chrome Side Panel on the currently active tab when you click the toolbar icon. No page content is read.
`contextMenus` Adds the "Analyze with PrepSnap" option to the right-click menu when you highlight text on any page. Only the text you have already selected is captured — nothing else on the page is read.
`storage` Saves your dark/light theme preference locally on your device only. Also used to temporarily pass selected text from the context menu to the side panel. No data is sent to any server via this permission.
`sidePanel` Enables PrepSnap to display its interface inside Chrome's native side panel rather than a popup. Required by Chrome for all side panel extensions.
PrepSnap does not request: tabs, webRequest, history, cookies, identity, downloads, geolocation, management, nativeMessaging, declarativeNetRequest, or access to all URLs (<all_urls>). The only host permission declared is https://*.supabase.co/* — solely to reach our own backend.

Third-Party Services

4.1 Groq API (AI Processing)

Job description content is sent to Groq's API to generate your interview pack. Groq processes this text to produce questions, culture insights, and checklist items.

  • Used only when you explicitly click "Generate Interview Pack"
  • No personally identifiable information is sent — only the job description text you provide
  • Groq does not retain API request data after processing per their data policy
  • Groq does not use your data for AI model training via the API

See Groq Privacy Policy →

4.2 Supabase (Backend Infrastructure)

Our edge function — which holds the Groq API key securely server-side — is hosted on Supabase. Job description content passes through Supabase's infrastructure as part of the API request and is not retained by Supabase on our behalf.

See Supabase Privacy Policy →

4.3 Google Fonts & Cloudflare CDN

The extension loads display fonts from Google Fonts and the jsPDF library from Cloudflare's CDN. These are standard web resources loaded for visual rendering and PDF generation. No personal data from you is transmitted as part of these requests beyond standard HTTP headers (e.g. IP address) that any web request involves.

User Control

  • All data processing is triggered only by your explicit action — clicking "Generate Interview Pack"
  • You can clear all locally stored data at any time via Chrome Settings → Extensions → PrepSnap → Clear storage
  • You can uninstall PrepSnap at any time — this removes all locally stored data from your device immediately
  • There is nothing stored on our servers to delete — because we store nothing

Data Security

  • All communication between the extension and our backend uses HTTPS/TLS encryption
  • The Groq API key is stored as a server-side environment secret in Supabase — it is never present in the extension's code, never transmitted to your browser, and never exposed to users
  • No centralised storage of any user-provided content
  • Local data is stored using Chrome's sandboxed storage.local API, accessible only to this extension

Data Retention

  • Job description content is discarded immediately after your interview pack is generated — no retention of any kind
  • No long-term storage of any content you provide, on our servers or anywhere else
  • Theme preference persists locally on your device until you uninstall the extension or clear its data manually

User Rights

Because we do not store any personal data or job description content on our servers, the standard data deletion and access rights are satisfied automatically:

  • Uninstalling PrepSnap removes all locally stored data from your device
  • You can clear local data via Chrome's extension management page at any time
  • There is no user account to delete — PrepSnap v1.0 does not require or create any user account
  • There is no data held by us to request, correct, or export
